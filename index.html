PARTS & FUNTIONS

MGA BAHAGI NG MOBILE PHONE AT ANG KANILANG PAG-ANDAR

  • "ANTENA": ay ginagamit upang tumanggap at magpadala ng frequency ng radyo. Ito ay inbuilt sa cabinet ng mobile phone. Ang mga ito ay tinatawag na inbuilt antenna.

  • "BATTERY": ay ang tanging pinagmumulan ng power supply sa isang mobile phone. Tatlong uri ng baterya ang kadalasang ginagamit sa mga mobile cell phone at Tablet

  • "BATTERY CHARGER": ilang uri ng Charger ang ginagamit para i-recharge ang baterya ng isang mobile cell phone. Kabilang dito ang mga normal na charger ng baterya, Travel Adapter (TA), Cigar Lighter Adapter (CLA) atbp.

  • "Switch o Key Tip": Ang maliliit na switch na ito ay ginagamit para sa key pad.

  • "CPU":Ang CPU ay ang Main Control Section ng isang mobile phone, Android Smartphone at Apple iPhone. Kinokontrol nito ang lahat ng function at ginagawa ang lahat ng gawain sa pagproseso.

  • "RAM": ay isang nabubura na memorya kung saan maaaring mabura ang mas lumang data at impormasyon at maiimbak ang bagong data at impormasyon.

  • "ROM": Isang alaala na mababasa lamang. Hindi ito mabubura. Ang Programmable ROM (PROM) at Erasable ROM (EROM) ay ilan pang uri ng ROM Memory na maaaring mabura.

  • "VISUAL DISPLAY": Ang bahaging ito ng mobile phone ay bumubuo ng liwanag upang ipakita ang lahat ng impormasyon. Ang pagpapakita ng isang cell phone ay maaaring uriin bilang mga sumusunod

  • "ANTENA SWITCH": Ginagawa nito ang function ng pagtanggap, pagpapadala at paglipat ng iba't ibang frequency tulad ng GSM, DCS at PCS.

  • "IF IC":Ang IC na ito ay tinatawag ding Hager IC, MJOLNER IC at RF IC. Ginagawa nito ang function ng amplification ng pagtanggap ng signal, paghahalo at demodulation

  • VCO (Voltage Control Oscillator): Kinokontrol ang dalas.

  • :AUDIO IC":Ang pangunahing tungkulin ng IC na ito ay upang hawakan at kontrolin ang dalas ng audio at makagawa ng tunog.

  • (Universal Energy Module): Gumaganap ito ng ilang function tulad ng power supply, charging, audio processing, ringer, vibrator atbp.

  • (Light Emitting Diode) Ito ay ginagamit upang makagawa ng liwanag sa keyboard at display.

  • "RF CRYSTAL": Ito ang pangunahing at napakahalagang kristal sa anumang mobile phone. Ang mga kristal na pangunahing 13 MHz, 26 MHz, 19.2 MHz o 36.4 MHz ay ​​ginagamit sa mga cell phone. Ang telepono ay magiging patay kung ang kristal na ito ay may sira.

  • :EAR SPEAKER": Ang Ear Speaker na ito ay naka-mount sa itaas na bahagi ng mobile phone. Nakakatulong itong makinig ng tunog sa mga papasok na tawag.

  • "MICOPHONE": Kino-convert nito ang voice signal sa electric signal o tone signal habang nagpapadala. Ipapadala ang signal na ito sa Audio IC.

  • VIBRATOR MOTOR Ito ay nagbibigay ng alerto sa panginginig ng boses sa panahon ng mga papasok na tawag o mensahe sa silent mode o kapag ang opsyon ay Naka-on.

  • "SIM card": SIM ay kumakatawan sa Subscriber Identity Module. Ito ay ginagamit sa mga GSM Mobile phone. Mayroong micro chip sa card na ito kung saan nakaimbak ang lahat ng data at impormasyong nauugnay sa USER. Kung walang SIM Card, walang magandang silbi ang GSM phone.

  • "MEMORY CARD": External memory storage card na ginagamit upang mag-imbak ng data tulad ng mga larawan, video, audio atbp. SD card, Mini SD Card at Micro SD card ay ginagamit bilang external memory.

  • "EXTERNAL ANTENNA SOCKET": Maaaring ikonekta ang panlabas na antenna sa tulong ng mga accessory sa socket na ito. Nakakatulong ito upang mapataas ang saklaw ng panloob na antenna.

  • ":RETU IC": Ang module na ito ay gumaganap ng ilang mahahalagang function.

  • :TAHVO IC": Ang modyul na ito ay gumaganap din ng ilang mahahalagang function

  • "OMAP MODULE": Nagsasagawa ng function ng pagpoproseso ng software ng application at kontrol ng UI software.

  • HANDS FREE CONNECTOR": Tumutulong na magpatuloy sa pag-uusap nang hindi hawak ang Telepono gamit ang Head Phone o Ear Phone

  • "FLASH IC": Ang memorya na ito sa loob ng telepono ay ginagamit upang mag-imbak ng mga program at hindi pabagu-bagong data

  • "RAP 3G IC": Ang IC na ito ay ginamit bilang core na may non Flash at SD RAM memory module sa modem section. Ang RAP3G module ay gumagana bilang cellular protocol na may WCDMA at GSM.

  • USB CONNECTOR": Ang connector na ito ay ginagamit upang mag-download ng data at para sa pag-charge ng Telepono.,

  • "JOY STICK": Ginagamit upang mapagaan ang paggana ng user. Maaaring kontrolin ng user ang pagpapatakbo ng isang cell phone sa pamamagitan ng pagtatakda ng mga function ng joystick.

  • "SOFTWARE CONNECTOR PARTS": Ginagamit para sa pag-flash ng Software o Operating System. Ang Telepono ay maaaring konektado sa isang Computer sa pamamagitan ng connector na ito upang malutas ang mga problemang nauugnay sa software.

  • "TIP NG CONNECTOR": Ginagamit upang i-interface ang Speaker, Ringer, Vibrator at Antenna.

  • "TESTING POINTS": Ang output ng iba't ibang uri ng signal ay sinusuri mula sa mga tip na ito sa tulong ng CRO.

  • "LTR": Ang regulator na ito ay ginagamit upang ayusin ang boltahe.
